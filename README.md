# SheCan Connect Portal

A modern NGO platform developed for She Can Foundation to support volunteer registration, community engagement, and women empowerment initiatives.

## Features

* Home Page
* About Section
* Programs Showcase
* Volunteer Registration Form
* Contact Form
* Admin Dashboard
* Dark Mode
* Animated Statistics Counters
* Toast Notifications
* Responsive Design
* Modern Glassmorphism UI

## Project Overview

SheCan Connect Portal is designed to help women connect with opportunities, training programs, and volunteer activities. The platform provides an engaging and user-friendly experience while promoting social impact and community participation.

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Local Storage
* Font Awesome Icons
* Responsive Web Design

## Modules

### Home

Displays organization mission, impact statistics, and call-to-action buttons.

### About

Provides information about the mission, vision, values, and impact of She Can Foundation.

### Programs

Showcases various programs including:

* Education
* Skill Development
* Health Awareness
* Leadership
* Entrepreneurship

### Volunteer Registration

Allows users to register as volunteers by submitting their personal details, skills, interests, and availability.

### Contact

Enables visitors to send messages and inquiries to the organization.

### Admin Dashboard

Displays volunteer registrations and contact messages for basic administration.
